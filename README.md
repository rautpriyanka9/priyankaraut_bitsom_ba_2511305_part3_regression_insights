# Task 1: Dataset Understanding

## Dataset Overview

The dataset contains monthly performance data for 80 retail stores across 4 months (320 observations total). The objective is to identify factors that influence monthly sales performance using regression analysis.

---

## Dependent Variable (Target Variable)

The dependent variable is:

| Variable        | Description                                                                                                      |
| --------------- | ---------------------------------------------------------------------------------------------------------------- |
| `monthly_sales` | Total monthly sales generated by each store. This is the target variable that the regression model will predict. |

---

## Potential Independent Variables (Predictors)

The following variables may influence monthly sales:

| Variable                     | Description                                                 |
| ---------------------------- | ----------------------------------------------------------- |
| `marketing_spend`            | Monthly marketing expenditure                               |
| `footfall`                   | Monthly visitor count                                       |
| `avg_discount_pct`           | Average discount percentage offered                         |
| `staff_count`                | Number of store employees                                   |
| `inventory_availability_pct` | Average inventory availability percentage                   |
| `competitor_distance_km`     | Distance to nearest competitor                              |
| `holiday_flag`               | Indicates whether the period included a significant holiday |
| `customer_rating`            | Average customer satisfaction rating                        |
| `region`                     | Business region                                             |
| `store_type`                 | Store category                                              |

---

---

## Variables Potentially Not Useful for Regression

### `store_id`

**Reason:**

* Unique identifier only.
* Does not provide explanatory business information.
* Should be excluded from regression modeling.

### `monthly_profit`

**Reason:**

* Highly related to sales and may introduce data leakage if predicting sales.
* Better suited as a secondary business KPI rather than a predictor.
